# Showroom (Widgetbook)

Component showcase using Widgetbook for the design system.

## Features

- Interactive component showcase
- Multi-brand theme preview
- Device frame preview
- Text scale testing
- Component variants and states

## Getting Started

### Installation

```bash
flutter pub get
```

### Running Showroom

```bash
flutter run
```

The Widgetbook app will launch showing all available components with different themes and configurations.

## Components Showcased

### BrandButton

- **Primary Button** - Main call-to-action style
- **Secondary Button** - Alternative action style
- **Outline Button** - Subtle action style
- **With Icon** - Buttons with icons
- **Sizes** - Small, Medium, Large
- **Loading State** - Loading indicator

## Themes Available

Switch between themes in the Widgetbook interface:

- Match
- Meetic
- OKCupid
- Plenty of Fish

## Device Frames

Preview components on different device sizes:

- iPhone 13
- Samsung Galaxy S20

## Usage

1. Launch the Showroom app
2. Select a component from the sidebar
3. Choose a use case
4. Switch themes using the theme addon
5. Test different device frames
6. Adjust text scale for accessibility testing
